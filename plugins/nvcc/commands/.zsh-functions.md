# .zsh-functions

Show all custom zsh functions from your minimal shell setup.

## Instructions

1. Read the ~/.zshrc file
2. Extract all function definitions (format: `name() { ... }` or `function name() { ... }`)
3. For each function, show:
   - Function name
   - What it does (brief description)
   - Usage example
   - Location in .zshrc (line number if helpful)

4. Group functions by purpose:
   - **Navigation**: d (cd to ~/code)
   - **Development**: cdg (cd to git root)
   - **Utilities**: x (extract archives), json/yaml (pretty print)
   - **System**: sudo-command-line (Ctrl+S toggle)
   - **NVM**: load-nvmrc (auto-switch Node versions)
   - **Prompt**: precmd, preexec (git branch + timing)

## Example Output Format

```
🔧 ZSH FUNCTIONS (Custom Shell Utilities)

NAVIGATION
  d()
    → Jumps to ~/code directory
    Usage: d

  cdg()
    → Changes to git repository root
    Usage: cdg (when inside a git repo)

UTILITIES
  x <archive>
    → Extracts any archive type (zip, tar, gz, etc.)
    Usage: x myfile.tar.gz

  json [file]
    → Pretty prints JSON (from stdin or file)
    Usage: curl api.com | json
    Usage: json data.json

...
```

These are YOUR custom functions, not Oh My Zsh plugins!
