# .zsh-aliases

Show all zsh aliases from your minimal, ADHD-friendly shell setup.

## Instructions

1. Read the ~/.zshrc file
2. Extract all lines starting with `alias`
3. Format them in a clean, easy-to-scan table
4. Group by category:
   - Navigation (d, de, c, etc.)
   - Modern replacements (ls, cat, etc.)
   - Git (lz)
   - Utilities (pg)
   - Aerospace (window manager)
   - Shell config (cz, sz)
   - Projects (p.\*)
   - Node/Dev (w.nvmrc)
   - Tmux (tx)
   - NPM/PNPM (ni, nr, nrs, etc.)

5. For each alias, show:
   - Alias name
   - Command it runs
   - Brief description

## Example Output Format

```
📋 ZSH ALIASES (Minimal Setup - No Oh My Zsh)

NAVIGATION
  d       → cd ~/code              Jump to code directory
  de      → cd ~/Desktop           Jump to Desktop
  c       → code .                 Open VS Code in current dir

MODERN REPLACEMENTS
  ls      → eza --color --icons    Better ls with colors
  ll      → eza -l ...             Long list format
  cat     → bat --paging=never     Syntax-highlighted cat

...
```

## Note

This reads from: `~/code/dotfiles/.zshrc` (symlinked from ~/.zshrc)
