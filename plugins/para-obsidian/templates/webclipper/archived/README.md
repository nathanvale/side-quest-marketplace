# Archived Web Clipper Templates

These templates were archived on 2026-01-13 as part of the ADHD-friendly Web Clipper simplification.

## Why Archived?

The original 15 specialized templates required too many decisions at capture time:
- Which template to use?
- Is this a recipe or an article?
- Is this a product or a gift idea?

This cognitive load contradicts the ADHD-friendly principle: **Capture now, organize later.**

## New Approach

A single `capture.json` template captures everything to the inbox. Classification happens during `para scan` where the clipping-strategy automatically detects content type from URL patterns and content markers.
