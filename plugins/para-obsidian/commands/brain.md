# Brain

Your second brain orchestrator. Routes natural language to the right para-obsidian skill. Say what you need, not which command.

## Usage

```
/para-obsidian:brain <anything vault-related>
```

## Examples

```
/para-obsidian:brain https://example.com
/para-obsidian:brain process my inbox
/para-obsidian:brain research this and make a resource https://youtube.com/watch?v=abc
/para-obsidian:brain review my day
/para-obsidian:brain create a project for GMS checkout refactor
/para-obsidian:brain save what we just discussed about orchestrator patterns
/para-obsidian:brain check SLO health
/para-obsidian:brain distill that resource on progressive summarization
```

## Instructions

When invoked, load the `brain` skill for full routing details:

```
@plugins/para-obsidian/skills/brain/SKILL.md
```
