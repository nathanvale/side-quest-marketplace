# The Layers

Progressive summarization uses multiple layers to compress information while preserving context.

| Layer | What | How | When |
|-------|------|-----|------|
| **0** | Original source | Full article/video/transcript | Already captured |
| **1** | Captured Notes | Raw notes, quotes, ideas | During triage |
| **2** | Bold Passages | **Bold** the most important parts | This skill |
| **3** | Highlighted Core | ==Highlight== the absolute essence | This skill |
| **4** | Executive Summary | 3-5 takeaways in YOUR words | This skill |

**Key insight:** Each layer preserves the previous ones. You can be bold with compression because the context is still there if needed.

**Compression Target:** Aim for 10-20% of original content to be bolded in Layer 2.
